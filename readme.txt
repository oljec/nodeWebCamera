Трансляція налаштована для роботи з локального сервера. Клієнт відповідно теж. Можна переналаштувати.

Кроки по запуску:
1) Запускається Node сервер з папки Server
2) Запускається файл для захоплення зображення з веб-камери клієнта index.html через Mozilla Firefox. При відкритті дозволяємо використовувати веб-камеру.
3) Починаємо трансляцію через кнопку на сторінці
4) Відкриваємо в браузері файл клієнта client.html
5) Дивимось трансляцію
